Example # Ecommerce-MusicStore
================================
Requirements
------------
* [Java Platform (JDK) 7]
(http://www.oracle.com/technetwork/java/javase/downloads/index.html)
*
 [Apache Maven 3.x](http://maven.apache.org/)


Quick start
-----------
1. `mvn package`

2. `java -jar target/example-spring-boot-rest-1.0-SNAPSHOT.jar`
